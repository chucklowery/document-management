# Feature AS-035 — Findability Evaluation

**Stories:** US-090
**Requirements:** REQ-IA-020, REQ-IA-021

## Scenario AS-035.1 — Material regression blocks adoption
**Given** a proposed navigation or labeling change is evaluated with representative tasks,
**When** it causes an unapproved material loss in task completion or accessibility,
**Then** adoption is blocked unless a time-bounded exception records scope, rationale, monitoring, and reversal criteria.

# Wayfinding Acceptance Specifications

# Feature AS-029 — Wayfinding and Complementary Paths

**Stories:** US-076, US-077  
**Requirements:** REQ-IA-001, REQ-IA-002, REQ-IA-003, REQ-IA-004

## Scenario AS-029.1 — User remains oriented
**Given** a user enters a Resource through search,  
**When** the Resource view opens,  
**Then** the view identifies scope, identity, Revision, time boundary, arrival path, and available search, browse, Relationship, backlink, and Trail paths.

## Scenario AS-029.2 — Resumed address preserves identity
**Given** a saved address identifies an exact Revision,  
**When** it is reopened after labels change,  
**Then** the same identity and Revision are resolved rather than a similarly labeled object.

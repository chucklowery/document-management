# Findability User Stories

**Project:** Organizational Knowledge and Work System

## US-076 — Orient Within a View
**As an information seeker,** I want to see active scope, current identity, Revision, constraints, and available paths, so that I know where I am.

## US-077 — Use Complementary Discovery Paths
**As an information seeker,** I want to move among search, browse, facets, Relationships, backlinks, Trails, and chronology, so that one ranking or hierarchy does not limit discovery.

## US-078 — Understand Labels
**As an information seeker,** I want labels, aliases, definitions, and owning context explained, so that I can interpret unfamiliar or ambiguous terms.

## US-079 — Refine Without Losing State
**As an information seeker,** I want applied filters, facets, ordering, corpus, and time boundary preserved and reversible, so that exploration remains understandable.

## US-080 — Recover From No Results
**As an information seeker,** I want spelling, alias, scope, and filter suggestions without silent broadening, so that I can recover deliberately.

## US-081 — Judge a Result Before Opening It
**As an information seeker,** I want a truthful preview with identity, authority, freshness, and limitations, so that I can choose efficiently.

## US-082 — Save and Resume a Workspace
**As a workspace owner,** I want to preserve selected identities, refinements, views, and a review checkpoint, so that I can continue later.

## US-083 — Control Personalization
**As an information seeker,** I want material personalization explained and resettable, so that adapted presentation remains under my control.

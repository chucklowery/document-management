# Findability Evaluation Requirements

## REQ-IA-020 — Task-Based Findability Evaluation
**EARS:** When a material Navigation Model, labeling, refinement, preview, or personalization change is proposed, the system shall evaluate it with a versioned Findability Evaluation Set and record task success, completion time, abandonment, recovery, accessibility, credibility comprehension, and applicable retrieval measures.
**Stories:** US-090

## REQ-IA-021 — Govern Findability Regression
**EARS:** If a proposed change causes an unapproved material findability or accessibility regression, then the system shall block adoption or require a time-bounded exception with scope, rationale, monitoring, and reversal criteria.
**Stories:** US-090

# Refinement Acceptance Specifications

# Feature AS-030 — Refinement and Recovery

**Stories:** US-078, US-079, US-080
**Requirements:** REQ-IA-005, REQ-IA-006, REQ-IA-007, REQ-IA-008

## Scenario AS-030.1 — Constraints remain visible
**Given** a Query has facets and a time boundary,
**When** results refresh,
**Then** each constraint remains visible and removable.

## Scenario AS-030.2 — Recovery requires a choice
**Given** a Query returns no results,
**When** the system suggests a wider search,
**Then** the proposed change is shown before it is applied.

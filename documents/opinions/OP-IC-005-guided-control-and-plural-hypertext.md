# Opinion Article OP-IC-005 — A Guided Workflow Is One Path Through the System

**Status:** Interpretive  
**Counterpoint:** Ted Nelson  
**Associated principles:** IC-P-001, IC-P-006, IC-P-007  
**Associated requirements:** REQ-IC-001, REQ-IC-007, REQ-IC-009, REQ-IC-017, REQ-IC-021

A Nelson-oriented counterpoint is that a wizard, dashboard, or recommended sequence can make one path appear to be the structure of the information itself.

The project should preserve links among configurations, source Revisions, examples, outputs, alternatives, prior paths, and related decisions. Users should be able to leave a guided flow, inspect exact sources and relationships, compare alternatives, and return without losing state.

Guidance may reduce effort, but it should not erase plural views or authored connections.

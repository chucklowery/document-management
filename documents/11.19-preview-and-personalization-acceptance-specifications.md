# Preview and Personalization Acceptance Specifications

# Feature AS-031 — Trustworthy Preview and Personalization

**Stories:** US-081, US-083
**Requirements:** REQ-IA-009, REQ-IA-010, REQ-IA-011, REQ-IA-012

## Scenario AS-031.1 — Preview preserves source context
**Given** a result preview supports a consequential choice,
**When** it is displayed,
**Then** it identifies source, Revision or time boundary, authority, freshness, and limitations.

## Scenario AS-031.2 — Personalization can be reset
**Given** personalization changes result order,
**When** the user requests an explanation or reset,
**Then** the applied policy and effect are shown and the unpersonalized view can be restored where permitted.

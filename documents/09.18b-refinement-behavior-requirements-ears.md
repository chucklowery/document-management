# Refinement Behavior Requirements

## REQ-IA-006 — Facet State
**EARS:** When facets are shown, the system shall identify applied values and show only permitted options and counts.
**Stories:** US-079

## REQ-IA-007 — Preserve Refinement
**EARS:** When a result view changes, the system shall preserve the Query, filters, facets, ordering, corpus, time boundary, and reversible interaction history.
**Stories:** US-079

## REQ-IA-008 — No Silent Broadening
**EARS:** If a Query returns no results, then the system shall not remove constraints, change scope, or substitute alternate terms without showing the proposed recovery action.
**Stories:** US-080
